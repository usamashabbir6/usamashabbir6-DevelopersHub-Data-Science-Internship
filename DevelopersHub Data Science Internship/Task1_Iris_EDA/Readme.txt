Task 1 — Exploring and Visualizing the Iris Dataset
Field Details 
Objective Understand how to load, summarize, and visualize a dataset
Dataset Iris Dataset (built into Seaborn)
Type Exploratory Data Analysis (EDA)

Approach:

Loaded the Iris dataset directly from Seaborn
Inspected shape, columns, and data types using pandas
Checked for missing values (none found)
Created scatter plot, histogram, and box plot visualizations

Key Insights:

Dataset contains 150 records across 3 species: Setosa, Versicolor, Virginica
Dataset is perfectly balanced with 50 samples per species
Setosa is clearly separable from the other two species based on petal size
Petal length and petal width are the most useful features for distinguishing species
Versicolor and Virginica overlap slightly in petal measurements

Skills Demonstrated:

Data loading and inspection using pandas
Basic data summarization with .describe() and .info()
Visualization using matplotlib and seaborn
